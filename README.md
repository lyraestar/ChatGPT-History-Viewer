# ChatGPT-History-Viewer
After downloading your chat history from ChatGPT Degrade Checker, you need a tool to revisit them as simple as in ChatGPT.  
English and Chinese are supported in GUI.

## Usage:
1. download and extract your chat history to a local folder by ChatGPT Degrade Checker, a web browser extention.  
2. open the local folder by chatgpt_history_viewer, and enjoy.
